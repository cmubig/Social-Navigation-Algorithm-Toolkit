# Social-Navigation-Algorithm-Toolkit
A collection of social navigation algorithms with minimal examples, easy integration with external system.

Currently supported algorithms:  
CADRL  
Socialforce  
RVO  
SPEC  
and more.

---
Usage
```bash
python example_cadrl.py
```
